Usemos datos de un JSON. Tales datos fueron extraídos de una tabla sobre Inmigrantes internacionales según región de residencia habitual y sexo. La tabla está incluida en el informe sobre Características de la inmigración internacional en Chile (INE, 2017).

Ahora estamos probando Chart.js, una biblioteca de JavaScript distinta de p5.js. Ya NO podemos cargar datos con un loadJSON(). Desde hoy usaremos la API Fetch de JavaScript.

El gráfico que sigue consulta datos de un GeoJSON. Tales datos describen todos los movimientos telúricos sobre 4.5M registrados por la USGS en el último mes.
